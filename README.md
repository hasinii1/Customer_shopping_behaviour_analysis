# 📌 Project Overview
This project focuses on understanding customer shopping behavior through data analysis, SQL querying, business intelligence dashboards, and interactive web applications.

The project transforms raw customer transaction data into meaningful insights that help businesses make data-driven decisions.

### 🎯 Objectives

- Analyze customer purchasing behavior
- Identify revenue-generating categories
- Understand customer demographics
- Evaluate subscription impact on spending
- Analyze shipping preferences
- Discover top-performing products
- Explore customer satisfaction through ratings

# 🚀 How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/customer-shopping-behaviour-analysis.git

cd customer-shopping-behaviour-analysis
```

## 2️⃣ Install Dependencies

```bash
pip install pandas numpy plotly streamlit sqlalchemy pymysql
```

## 3️⃣ Run Streamlit Dashboard

```bash
streamlit run app.py
```

## 4️⃣ Execute SQL Queries

```sql
USE customer_behaviour;
```

Run all business queries in MySQL Workbench.
